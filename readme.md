# 2025 新春版：一个近乎完美的 Konva 手写板诞生记

基于 Konva 开发的可直接用于各类项目的 Web 手写板组件。

包含 4 套代码：

1. React18/19(javascript)
2. React18/19(typescript)
3. Vue3(javascript)
4. Vue3(typescript)

## 目录结构说明

dist 目录为 build 后的代码，可自行下载体验最终效果。

zip 压缩包为项目源码，`解压密码，见配套教程原文`。
